# Community Health Survey Form

## Project Overview
Community Health Survey Form is a Flutter-based mobile application developed using the Dart programming language. The application is designed to collect, store, and manage community health survey data efficiently.

It supports offline data storage using SQLite (sqflite), making it suitable for field surveys where internet connectivity may not always be available.

---

## Objectives
- Collect health-related information from community members
- Store survey data locally using SQLite (sqflite)
- Provide an easy-to-use and user-friendly interface
- Enable offline data entry and management

---

## Technologies Used
- **Flutter**
- **Dart**
- **SQLite (sqflite)**
- **Material UI**

---

## Features
- User-friendly health survey form
- Offline data storage using sqflite
- Structured data management
- Reliable and efficient local database usage
